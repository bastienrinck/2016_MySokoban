# 2016_MySokoban

Sokoban game under terminal.

## Usage
```
./my_sokoban [map file]
```

## Map exemple
```
#################
#               #
# O  P          #
#               #
# X      X      #
#               #
#  O            #
#################
```
```
 P: Player
 X: Boxes
 O: Target holes
```
